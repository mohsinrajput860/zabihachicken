# Zabiha Chicken - Restaurant Website

Zabiha Chicken is a modern, fully responsive restaurant website built with PHP and MySQL. It features a sleek design, a dynamic menu, an admin dashboard for management, and sections for catering and franchise inquiries.

## 🚀 Features

### Frontend
- **Dynamic Hero Section**: Interactive banner slideshow showcasing top dishes.
- **Menu Management**: Browse full categories of food items.
- **Popular Items Carousel**: Automatically scrolling carousel of top-rated dishes.
- **Catering Services**: Dedicated page for catering inquiries and management.
- **Gift Cards**: Information and options for purchasing gift cards.
- **Careers/Franchise**: Page for franchise applications.
- **SEO Optimized**: Built-in SEO management for better search rankings.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

### Admin Dashboard
- **Secure Login**: Protected admin access.
- **Manage Specials**: Add, edit, or remove daily specials and offers.
- **Catering Management**: View and manage catering requests.
- **Franchise Applications**: Track and manage interested franchise partners.
- **SEO Settings**: Easily update meta titles and descriptions for each page.
- **User Management**: Manage admin users and permissions.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap.
- **Backend**: PHP (PDO for secure database interactions).
- **Database**: MySQL.

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/zabihachicken.git
   ```

2. **Database Setup**:
   - Open XAMPP/WAMP and start Apache and MySQL.
   - Create a database named `Zabiha-Chicken` in phpMyAdmin.
   - Import the SQL file located in the `databse/` directory.

3. **Configuration**:
   - Open `config/config.php`.
   - Update the `DB_HOST`, `DB_NAME`, `DB_USER`, and `DB_PASS` constants if they differ from your local setup.
   - Update `$base_url` to match your local project path (e.g., `http://localhost/zabihachicken/`).

4. **Access the Website**:
   - Visit `http://localhost/zabihachicken/` in your browser.
   - For Admin access, visit `http://localhost/zabihachicken/admin/login.php`.
